## 使用并行流处理大规模数据集：MapReduce 模型
Java 8 引入的最重要的创新是 lambda 表达式和流 API。  

流是可以以顺序方式或者并行方式处理的一个元素序列。  
可以应用中间操作转换流，然后执行最终计算以获得预期结果（列表、数组、数值等）。

### 小结
-	流是一个数据序列（并不是一个数据结构）  
允许以顺序或者并发方式应用一个操作序列，对元素进行筛选、转换、排序、约简或组织，以获得一个最终对象。
